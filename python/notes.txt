-------------------------------------------------

examples:  book = {"chap1":10 ,"chap2":20 ,"chap3":30}
           data = {1:2,3:4,5:6}

working with flat files 
-----------------------
read mode - r 
write mode - w 
append mode - a

opening the file 
----------------
syntax:  object = open(filename,mode )
example:    fr = open("clients.txt","r") # read mode 
            case1:  if the file is existing .. then only we will be able to read the file

            fw = open("clients.txt","w") # write mode 
            case1: if the file is not found ... file gets created first 
            case2: if the file is already existing.. it overwrites the existing data

            fa = open("clients.txt","a") # append mode 
            case1: if the file is not found ... file gets created first 
            case2: if the file is already existing.. it adds the new data to the existing data
