# Ecommerce Logistic Regression Classification Project

## Files Included
- `ecommerce_classification_data.csv` - synthetic ecommerce dataset
- `logistic_regression_ecommerce_classification.ipynb` - Jupyter notebook
- `streamlit_app.py` - Streamlit application
- `requirements.txt` - Python dependencies

## Install
```bash
pip install -r requirements.txt
```

## Run Jupyter Notebook
```bash
jupyter notebook logistic_regression_ecommerce_classification.ipynb
```

## Run Streamlit App
```bash
streamlit run streamlit_app.py
```
